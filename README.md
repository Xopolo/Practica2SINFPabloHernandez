**Práctica 2 Sistemas de informacion - Tratamiento paralelo de datos utilizando map-reduce**

Practica desarrollada bajo java 8 con dependencias especificadas en [pom.xml](https://github.com/Xopolo/Practica2SINFPabloHernandez/blob/d5c8390fe64fd403846ebfb6282ad096025ff6ff/README.md)
en las que se incluyen las necesarias para utilizar [hadoop](https://hadoop.apache.org/) en remoto

La carpeta ```CarpetaSSH```, contiene los resultados obtenidos en las consultas así como los scripts utilizados para 
facilitar el proceso de ejecucion

La clase ``ConsultaManager``, es la que gestiona que consulta se va a ejecutar, mientras que los paquetes
``Consulta{1-7}`` contienen las clases Mapper y Reducer necesarias para cada consulta
